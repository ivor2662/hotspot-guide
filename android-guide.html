Android 熱點設定教學

1. 打開「設定」App
2. 點選「網路與網際網路」或「連線」
3. 點選「個人熱點」或「行動無線基地台」
4. 開啟「行動熱點」開關
5. 確認熱點名稱與密碼，刷卡機將自動連線

各品牌設定說明

設定 → 連接 → 行動無線基地台與網路共用 → 行動無線基地台 → 開啟

設定 → 可攜式熱點 → WLAN 可攜式熱點 → 開啟

設定 → 其他無線連接方式 → 個人熱點 → Wi-Fi 熱點 → 開啟

設定 → 網路與網際網路 → 熱點與網路共用 → Wi-Fi 熱點 → 開啟

設定 → 網路與網際網路 → 熱點與網路共用 → 可攜式 Wi-Fi 熱點 → 開啟

設定 → 網路與網際網路 → 熱點與網路共用 → Wi-Fi 熱點 → 開啟
