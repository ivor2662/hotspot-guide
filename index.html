請依照以下步驟開啟手機的個人熱點

📱 Android 裝置

1. 打開「設定」App
2. 點選「網路與網際網路」或「連線」
3. 點選「個人熱點」或「行動無線基地台」
4. 開啟「行動熱點」開關
5. 確認熱點名稱與密碼，刷卡機將自動連線

🍎 iPhone 裝置

1. 開啟「設定」App
2. 點選「個人熱點」
3. 開啟「允許其他人加入」
4. 確認熱點密碼，並等待刷卡機連線

📌 各品牌 Android 手機熱點設定參考

Samsung（三星）

設定 → 連接 → 行動無線基地台與網路共用 → 行動無線基地台 → 開啟

Xiaomi（小米 / 紅米）

設定 → 可攜式熱點 → WLAN 可攜式熱點 → 開啟

OPPO / realme / OnePlus

設定 → 其他無線連接方式 → 個人熱點 → Wi-Fi 熱點 → 開啟

ASUS（ZenFone）

設定 → 網路與網際網路 → 熱點與網路共用 → Wi-Fi 熱點 → 開啟

Sony Xperia

設定 → 網路與網際網路 → 熱點與網路共用 → 可攜式 Wi-Fi 熱點 → 開啟

Google Pixel（原生 Android）

設定 → 網路與網際網路 → 熱點與網路共用 → Wi-Fi 熱點 → 開啟

如無法找到設定，請參考手機品牌官方說明或聯繫客服協助。
